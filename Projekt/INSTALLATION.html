Installationsanleitung

Voraussetzungen

  • Internetverbindung
  • Browser
  • Ubuntu 24.04(DESKTOP)
  • Docker + Docker Compose installiert
  • Ports freigegeben: 8080, 8181, 8081, 9000

Installation

  1. Projekt clonen:
    git clone https://github.com/Svensemann93/LB-Projekt-M347.git
  2. In die Projekt-Ordner wechseln:
    cd Projekt/wordpress
  3. Stack starten:
    docker compose up -d
  4. Prüfen im Browser:
    • Wordpress: http://localhost:8080
    • MediaWiki: http://localhost:8181
    • Portainer: http://localhost:9000
    • Jira: http://localhost:8081
  5. Stack stoppen:
    docker compose down

Benutzer / Passwörter

  • MariaDB Wordpress → User: wordpress / Passwort: wordpresspass
  • MariaDB MediaWiki → User: wikiuser / Passwort: wikipass
  • Postgres Jira → User: jira / Passwort: jirapass
  • Portainer Admin → beim ersten Start selbst vergeben
